Desenvolvido em R

# Instalar o VEIN
install.packages("vein")
library(vein)

# Estimar emissões
emisions <- emis(
  veh = "PC", 
  size = "<1400cc", 
  fuel = "G", 
  pollutant = "CO",
  speed = speed_data
)

plot(emisions, col="#00FF88")

Por que usar o VEIN?

🚗

Integração de Tráfego

Leia matrizes de dados de tráfego, aplique simulações de tráfego e una dados espaciais de forma contínua.
