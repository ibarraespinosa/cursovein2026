# Módulo 2: Onde rodar?

Instalar R e RStudio no seu computador é uma ótima ideia, mas não é obrigatório para este curso. Vou te mostrar algumas alternativas que funcionam direto no navegador, sem precisar instalar nada.

Tu pode rodar onde quiser, mas vou te mostrar algumas alternativas, e nao precisa nada, so precisa internet!

## 1 [https://webr.r-wasm.org/latest/](https://webr.r-wasm.org/latest/)

![](content/figs/vein_wasm.png)

Na pagina web https://webr.r-wasm.org/latest/ voce pode rodar R no navegador. E para instalar o model temq ue rodar:

```r
webr::install("vein")
```

## 2 Instalando R no computador

Baixar R em [https://cran.r-project.org/bin/](https://cran.r-project.org/bin/)

Se quiser baixar R, precisa baixar tambem as ferramentas de compilacao:

- Windows: [rtools](https://cran.r-project.org/bin/windows/Rtools/)
- Mac: [xcode](https://developer.apple.com/xcode/)
- Linux: [r-base-dev](https://cran.r-project.org/bin/linux/)
